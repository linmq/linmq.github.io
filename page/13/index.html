Unity3D界面上Inspector的Rotation应该是相对于世界坐标的

我们知道在三维中旋转某个轴，该轴是不动的，只是另外两个轴组成的平面在动，这个说法是指绕着自身坐标系旋转，即局部坐标系（Local Coordinate）。一直以来，我以为Unity3D中的界面Inspector上的Rotation是绕着局部坐标系旋转，今天要旋转一个物体时，发现应该是世界坐标系。

https://cloudup.com/iX9n_Wr3b9X

当Rotation的X、Y、Z都是0的时候，你旋转某个轴是看不出来，但X、Y、Z上都已经有些值时，你就可以看出来它不是绕你想要的局部坐标系的轴在旋转。
把左上方的Local改为World，就能更好地看出想要的旋转。

https://cloudup.com/iwM61FmzC4y

xcode5误删文件的恢复办法

最近尝试在xcode下使用git.
创建工程时勾选Git是会在自动把所有文件提交。然后我不小心删除了某几个文件，但并没有commit。我想恢复这几个文件，该怎么做呢？
xcode5虽然集成了强大的git功能，但这个功能好像没有提供（后来研究了下，其实菜单里的Source Control / Discard All Changes 就可以撤消所有修改），我只好尝试敲命令。

第一个坑：Mac的终端，敲cd命令时，后面得跟上完整的路径，比如得这么敲 cd /Users/xiaoming/Document/Project/GitTestProject 这样才进得了目录。
而在windows下安装msysgit后的bash命令里却只需要敲下一级目录就可以直接进入下一级目录。当然Windos下的Dos的cd命令是一样的。

第二个坑：这个是自己没研究清楚。git reset –hard HEAD^ 这个命令是切换到当前仓库版本的前一版本。而我目前仓库里只有一个版本，那就相当于要切到-1版本去，那是一个不存在的版本，然后就报错了，我也看不懂那个错误。后来请教了网友 @偶尔e网事 才解决。

Unity3D材质丢失的解决办法

Win7 Unity3D 4.2.0f4
同事给我一些Fbx格式的角色动画，包括几个图片文件夹，和一个Materials文件夹。但我在导入Unity3D时，但主角的材质却丢失。
解决办法是把Materials文件夹删掉，然后那个文件夹上右键选择reimport，即可解决问题。
